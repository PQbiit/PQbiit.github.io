My Work

HeatStress

App aiming to lower the deaths of field workers by heat stroke related illnesses based on OSHA standards, the app calculates the individual heat stroke risk of a person by measuring his current location weather conditions and his own vital signs, providing him with reminders and recommendations to avoid having a heat illness. Using Java and SQLite, I was in charge of implementing the Wunderground API by IBM that provides weather data to the app, the algorithm that calculates the heat stroke risk of a person and the initial version of the app database.

ARCA CONTINENTAL - ANOMALY DETECTION IN SUPPLY CHAIN

Anomaly detection model developed for Arca Continental, Monterrey used to monitor the Key Performance Indicators (KPIs) of some of the company’s main products. Using data analysis techniques such as data clustering and time series forecasting, I developed an anomaly detection model that compares the time series forecasted data to the expected data for the KPIs which is then compared to a confidence threshold, identifying the record that may present an anomaly. This model was developed using both Python (Anaconda Jupiter Notebook) and R.

TECNOLÓGICO DE MONTERREY - CONEXIÓN TEC WEB SITE

Web system made to help the Tecnológico de Monterrey’s “Conexión Tec” entrepreneurship event management. This event seeks to bring students with innovative projects to experts in their respective industry or area of interest. Using Ruby on Rails, I was in charge of implementing the business logic of this event for the user, project and assistance management modules as well as refactoring the internal structure of the project using design patterns.
